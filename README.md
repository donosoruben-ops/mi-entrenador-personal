# Mi Entrenador Personal Pro

Aplicación web progresiva (PWA) para seguimiento de entrenamientos con planificación dinámica semanal.

## Características

- 📅 Calendario real integrado
- 🔄 Planificación semanal dinámica (ejercicios diferentes cada semana)
- 🎥 Videos de YouTube integrados para cada ejercicio
- 🏋️ Base de datos de ejercicios con equipamiento de CrossFit
- 📱 Diseño 100% responsive para móvil
- 💾 Persistencia de datos en localStorage

## Horarios de Entrenamiento

- **Lunes, Miércoles, Viernes**: 4:00-5:30 PM (pesas) + 6:00-7:00 PM (CrossFit)
- **Martes y Jueves**: 8:30-9:00 PM (30 min - brazos y pectorales)
- **Fin de semana**: Descanso

## División Muscular

- **Lunes**: Bíceps y Tríceps
- **Miércoles**: Pecho y Espalda
- **Viernes**: Piernas
- **Martes/Jueves**: Brazos y Pectorales

## Tecnologías

- HTML5
- Tailwind CSS
- JavaScript vanilla
- LocalStorage para persistencia

## Cómo usar

1. Abre la app en tu navegador
2. El calendario muestra los días de entrenamiento
3. Haz clic en cualquier día para ver su plan
4. Presiona "Generar Nueva Semana" para cambiar los ejercicios
5. Cada ejercicio tiene un link a YouTube para ver la técnica
